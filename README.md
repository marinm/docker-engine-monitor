# Docker Engine Monitor

## About

Browser client to the [Docker Engine API](https://docs.docker.com/engine/api).

Made for local instance monitoring, not remote.

Why I made this:
- to learn React.js
- to learn Docker
- to learn REST-ful API design

## Setup

Set the `HOST` and `PORT`.
